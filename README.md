# roblox-monitor

Choose documentation language:

- Bahasa Indonesia: [README.id.md](README.id.md)
- English: [README.en.md](README.en.md)

Main runtime is in `main.lua`.
`start.sh` is used as bootstrap/update + dependency installer, then it runs `main.lua`.

Quick start:

```sh
wget -O start.sh https://raw.githubusercontent.com/xnnnsets/roblox-monitor/refs/heads/main/start.sh && chmod +x start.sh && ./start.sh
```

Daily run:

```sh
~/roblox-monitor/start.sh
```
